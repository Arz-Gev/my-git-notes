1 - Better to have a global config with your name and email.

2 - Run git add . to add all changes to staging, or git add "file name" to add only the required ones. Then, commit them with a message using git commit -m "your message".

3 - If you make a mistake, Git will usually let you know, just read the messages carefully.

4 - Use branches.

5 - Try to commit regularly.
